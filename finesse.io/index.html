To rebind your controls, first export the settings by pressing [*]. Then update the json file (the values in the "keybinds" section). When you're done, save the file and drag & drop it into this page.
Click anywhere in the game screen to reset your current progress.
To modify any of the settings of the game, first export the settings by pressing [*] on your keyboard.
Open up the downloaded file on your text editor of choice, and set the values as you see fit.
Once you're done modifying the settings file, save it, then drag and drop it into this page, the settings will be imported automatically.
* Note that this program doesn't store any files in your system so every time you reload this page, you may need to re-import your settings.
GLOSSARY
Term Definition
DAS Delayed Auto-Shift: DAS refers to the delay before the Auto-Shifting occurs.
ARR Auto Repeat Rate: refers to the rate at which shifting automatically occurs.
If the value is -1, the piece will move against the wall immediately after shifting is triggered.
A change in direction will cause this to be reset.
SDR Soft-Drop Rate: refers to the rate at which the piece moves down ONCE.
If the SDR value is set to 0, the piece will move down every frame; if the value is set to -1 it will move instantly to the bottom.
All the finesse moves are based on the 0G 60Hz SRS Finesse listed on the harddrop wiki.
